\subsection{Detailed Description}
\mbox{\hyperlink{classshaderutils_1_1_shader_proxy}{Shader\+Proxy}} é uma classe que facilita criar programas gráficos em C++ e Open\+GL 2.\+1, reduzindo a quantidade de código necessário para conectar dados entre o programa em C++ e o código em G\+L\+SL. Crie um objeto Shade\+Proxy do seguinte modo\+: {\ttfamily  \mbox{\hyperlink{classshaderutils_1_1_shader_proxy}{Shader\+Proxy}} $\ast$proxy = new Shader\+Proxy(id\+\_\+programa, quantidade\+De\+Atributos); } onde\+: id\+\_\+programa é o identificador do programa gerado com a função gen\+Program e quantidade\+De\+Atributos é a quantidade de atributos que o programa G\+L\+SL possui.
